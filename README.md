# Python Project Template
> This is the starting point for my python projects

## Getting Started
1. **Clone the repository**
```bash
git clone https://github.com/aditya-shriwastava/py_project_template.git
cd py_project_template
```

2. **Create and activate a virtual environment**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. **Install dependencies**
```bash
pip install .
```
Or, for development (with extra tools):
```bash
pip install -e .[dev]
```

4. **Usage**
After installation, you can run app with:
```bash
app
```
